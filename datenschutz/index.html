Diese Website nutzt aus Sicherheitsgründen und zum Schutz der Übertragung vertraulicher Inhalte eine SSL- bzw. TLS-Verschlüsselung. Diese Website speichert Ihre gewählte Sprache (Deutsch oder Englisch) im sogenannten localStorage Ihres Browsers, damit die Seite bei einem erneuten Besuch in derselben Sprache angezeigt wird. Es werden keine personenbezogenen Daten an uns übermittelt; die Information bleibt ausschließlich in Ihrem Browser. Sie können diese Speicherung jederzeit über die Browser-Einstellungen löschen.

Wir setzen keine Cookies, keine Analyse-Tools, keine Tracker und keine Marketing-Pixel ein.

VI. Lokal eingebundene Schriftarten

Diese Website nutzt die Schriftarten Plus Jakarta Sans und Aleo, die lokal auf demselben Server gehostet werden, der diese Seite ausliefert. Es werden keine Verbindungen zu externen Servern (insbesondere nicht zu Google Fonts) für den Abruf von Schriftarten hergestellt. Somit findet in diesem Zusammenhang keine Übertragung personenbezogener Daten an Dritte statt.
